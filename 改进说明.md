## 技术细节

### 原生桥接工作原理
```javascript
// 网站端调用
window.NativeApp.takePhoto();

// 应用端接收
window.addEventListener('message', (event) => {
  if (event.data.type === 'TAKE_PHOTO') {
    // 调用原生相机
  }
});

// 应用端返回结果
webview.contentWindow.postMessage({
  type: 'PHOTO_TAKEN',
  data: { path: '/path/to/photo.jpg' }
}, '*');
```
