## 第四步：填写应用信息

### 4.1 基本信息

在 "App 信息" 页面填写：

- **名称**：齐家婚姻
- **副标题**（可选）：婚姻家庭服务平台
- **类别**：
  - 主要类别：生活
  - 次要类别：社交（可选）

### 4.2 定价与销售范围

- **价格**：免费
- **销售范围**：选择中国或全球

### 4.3 App 隐私

必须填写隐私政策 URL，例如：
- `https://qjmarriage.com/m/about.php?t=clause`（需要你在网站上创建这个页面）

隐私政策页面必须包含：
- 收集哪些数据
- 如何使用数据
- 是否分享给第三方
- 用户权利

### 4.4 准备提交

点击左侧 "1.0 准备提交"，填写：

#### App 预览和截图

需要准备不同尺寸的 iPhone 截图：

**必需尺寸：**
- 6.7 英寸显示屏（iPhone 15 Pro Max）：1290 x 2796 像素
- 6.5 英寸显示屏（iPhone 14 Plus）：1284 x 2778 像素
- 5.5 英寸显示屏（iPhone 8 Plus）：1242 x 2208 像素

每个尺寸至少 1 张，最多 10 张。

**截图建议：**
- 展示应用主要功能
- 清晰、高质量
- 不要包含设备边框
- 可以添加文字说明

#### 宣传文本（可选）

最多 170 个字符，例如：
```
专业的婚姻家庭服务平台，为您提供便捷的在线咨询服务。
```

#### 描述

最多 4000 个字符，例如：
```
齐家婚姻是一个专业的婚姻家庭服务平台。

主要功能：
• 在线咨询服务
• 专业的婚姻指导
• 便捷的预约系统
• 安全的隐私保护

我们致力于为用户提供优质的婚姻家庭服务，帮助每个家庭更加和谐美满。
```

#### 关键词

最多 100 个字符，用逗号分隔，例如：
```
婚姻,家庭,咨询,服务,情感,关系
```

#### 支持 URL

必填，例如：
```
https://qjmarriage.com/kefu/
```

#### 营销 URL（可选）

例如：
```
https://qjmarriage.com
```

## 审核时间线

- **提交后**：状态变为 "等待审核"
- **1-3 天**：Apple 开始审核，状态变为 "正在审核"
- **审核通过**：状态变为 "准备销售"，应用上架
- **审核被拒**：会收到邮件说明原因，修改后重新提交

---

## 常见审核被拒原因

### 1. 纯 WebView 应用

Apple 对纯 WebView 应用审核较严，建议：
- 在描述中说明应用提供的独特价值
- 确保网站针对移动端优化良好
- 添加一些原生功能（如推送通知）

### 2. 隐私政策

- 必须有隐私政策页面
- 必须说明数据收集和使用方式
- URL 必须可访问

### 3. 功能不完整

- 确保所有功能正常工作
- 测试账号必须有效
- 不能有明显的 bug

### 4. 元数据问题

- 截图必须真实反映应用功能
- 描述不能有误导性内容
- 不能包含其他平台的信息（如 Android）

---

## 审核通过后

1. 应用会自动上架（如果选择了自动发布）
2. 用户可以在 App Store 搜索并下载
3. 可以在 App Store Connect 查看下载数据和用户评价

---

## 更新应用

当需要更新时：

1. 修改代码
2. 推送到 GitHub（自动构建并上传）
3. 在 App Store Connect 创建新版本
4. 选择新的构建版本
5. 填写 "此版本的新增内容"
6. 重新提交审核

如果审核被拒或遇到问题：

4. 重新提交审核
