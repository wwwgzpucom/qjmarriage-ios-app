# QJ Marriage iOS 应用使用说明

## 📱 这是什么？

这是一个简单的 iOS WebView 应用，可以把你的网站包装成 iOS 应用，类似 Android SmartWebView。

---

## 🚀 快速开始（3步）

### 1️⃣ 修改网站地址

打开 `src/js/app.js` 文件，修改第 4 行：

```javascript
const WEBSITE_URL = 'https://你的网站.com';
```

把 `https://www.example.com` 改成你的网站地址。

### 2️⃣ 构建项目

在项目目录运行：

```bash
npm run build
npx cap sync
```

### 3️⃣ 在 Xcode 中打开

```bash
npx cap open ios
```

然后在 Xcode 中点击运行按钮（▶️）。

---

## 💻 环境要求

### Windows 用户（你当前的系统）

**重要提示**：iOS 应用只能在 macOS 上构建和测试！

你需要：
1. 一台 Mac 电脑（或者使用云端 Mac 服务）
2. 安装 Xcode（从 Mac App Store 免费下载）
3. 安装 CocoaPods：`sudo gem install cocoapods`

### 如果没有 Mac 怎么办？

可以使用这些服务：
- **MacStadium**：云端 Mac 租赁
- **MacinCloud**：按小时租用 Mac
- **GitHub Actions**：免费的 macOS 构建环境

---

## 📝 详细配置

### 修改应用名称

编辑 `capacitor.config.json`：

```json
{
  "appName": "你的应用名称"
}
```

### 修改包名（Bundle ID）

已设置为：`com.qjmarriage.app`

如需修改，编辑 `capacitor.config.json`：

```json
{
  "appId": "com.你的公司.应用名"
}
```

### 修改主题颜色

编辑 `src/index.html`，找到：

```html
<meta name="theme-color" content="#31d53d" />
```

改成你喜欢的颜色。

## 📱 上架 App Store 步骤

### 1. 准备工作

- [ ] 注册 Apple Developer 账号（$99/年）
- [ ] 准备应用图标（1024x1024px）
- [ ] 准备应用截图（不同尺寸的 iPhone）
- [ ] 编写应用描述
- [ ] 准备隐私政策网页

### 2. 在 Xcode 中配置

1. 打开项目：`npx cap open ios`
2. 选择 `App` 目标
3. 在 `Signing & Capabilities` 中：
   - 选择你的开发团队
   - 确认 Bundle Identifier：`com.qjmarriage.app`
4. 添加应用图标（在 `Assets.xcassets` 中）

### 3. 构建和上传

1. 在 Xcode 中选择 `Product > Archive`
2. 等待构建完成
3. 点击 `Distribute App`
4. 选择 `App Store Connect`
5. 上传到 App Store

### 4. 在 App Store Connect 中提交

1. 访问 https://appstoreconnect.apple.com
2. 创建新应用
3. 填写应用信息
4. 上传截图
5. 提交审核

## ⚠️ 重要提示

### App Store 审核要点

苹果对纯 WebView 应用审核很严格，建议：

1. **添加原生功能**：
   - 推送通知
   - 离线功能
   - 原生导航

2. **提供独特价值**：
   - 不能只是网站的镜像
   - 需要针对移动端优化

3. **完善的元数据**：
   - 清晰的应用描述
   - 高质量截图
   - 隐私政策

### 常见问题

**Q: 我在 Windows 上能测试吗？**  
A: 不能。iOS 应用只能在 Mac 上构建和测试。

**Q: 需要付费吗？**  
A: 上架 App Store 需要 Apple Developer 账号（$99/年）。

**Q: 审核需要多久？**  
A: 通常 1-3 天，首次可能更长。

**Q: 如何添加推送通知？**  
A: 需要配置 Firebase 或 Apple Push Notification Service (APNs)。
