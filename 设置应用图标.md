# 设置应用图标指南

## 📱 图标文件位置

你的应用图标 `appIcon.png` 已经放在项目中：
- `qjmarriage-app/appIcon.png` - 项目根目录
- `qjmarriage-app/src/assets/appIcon.png` - 资源目录

---

## 🎨 iOS 应用图标要求

### 图标尺寸要求

iOS 应用需要多个尺寸的图标：

| 用途 | 尺寸 | 说明 |
|------|------|------|
| App Store | 1024x1024 | 必需，用于商店展示 |
| iPhone | 180x180 | iPhone 主屏幕 (@3x) |
| iPhone | 120x120 | iPhone 主屏幕 (@2x) |
| iPad Pro | 167x167 | iPad Pro 主屏幕 |
| iPad | 152x152 | iPad 主屏幕 |
| 通知 | 40x40, 60x60, 80x80 | 通知图标 |
| 设置 | 58x58, 87x87 | 设置页面图标 |

### 图标设计规范

- ✅ **格式**：PNG（无透明度）
- ✅ **形状**：方形（系统会自动添加圆角）
- ✅ **颜色**：RGB 色彩空间
- ❌ **不要**：添加圆角、阴影、透明背景

## 🛠️ 方法 1：使用在线工具生成（推荐）

### 步骤 1：生成所有尺寸

访问以下任一网站，上传 `appIcon.png`：

1. **App Icon Generator**
   - 网址：https://www.appicon.co/
   - 上传图片 → 选择 iOS → 下载

2. **MakeAppIcon**
   - 网址：https://makeappicon.com/
   - 上传图片 → 生成 → 下载

3. **Icon Kitchen**
   - 网址：https://icon.kitchen/
   - 上传图片 → 选择平台 → 下载

### 步骤 2：解压下载的文件

你会得到一个包含所有尺寸的文件夹，例如：
```

├── Icon-App-29x29@1x.png
├── Icon-App-29x29@2x.png
├── Icon-App-29x29@3x.png
├── Icon-App-40x40@1x.png
├── Icon-App-40x40@2x.png
├── Icon-App-40x40@3x.png
├── Icon-App-60x60@2x.png
├── Icon-App-60x60@3x.png
```

## 🎯 方法 2：在 Xcode 中手动设置

### 步骤 1：打开 Xcode

在 Mac 上运行：
```bash
cd qjmarriage-app
npx cap open ios
```

### 步骤 2：找到 Assets

1. 在 Xcode 左侧项目导航器中
2. 展开 `App` 文件夹
3. 点击 `Assets.xcassets`
4. 点击 `AppIcon`

### 步骤 3：添加图标

有两种方式：

#### 方式 A：拖放图片（简单）
1. 准备好所有尺寸的图标
2. 直接拖放到对应的方框中

#### 方式 B：使用单张图片（Xcode 14+）
1. 删除所有现有图标
2. 在 `AppIcon` 设置中
3. 勾选 "Single Size"
4. 拖入 1024x1024 的图标
5. Xcode 会自动生成所有尺寸

### 步骤 4：验证

1. 选择一个模拟器（如 iPhone 15）
2. 点击运行按钮（▶️）
3. 查看主屏幕上的应用图标

## 📝 方法 3：使用命令行工具

### 安装 ImageMagick

在 Mac 上：
```bash
brew install imagemagick
```

### 生成所有尺寸

创建脚本 `generate-icons.sh`：

```bash
#!/bin/bash

# 输入图片
INPUT="appIcon.png"
OUTPUT_DIR="ios/App/App/Assets.xcassets/AppIcon.appiconset"

# 创建输出目录
mkdir -p "$OUTPUT_DIR"

# 生成各种尺寸
convert "$INPUT" -resize 20x20 "$OUTPUT_DIR/Icon-App-20x20@1x.png"
convert "$INPUT" -resize 40x40 "$OUTPUT_DIR/Icon-App-20x20@2x.png"
convert "$INPUT" -resize 60x60 "$OUTPUT_DIR/Icon-App-20x20@3x.png"
convert "$INPUT" -resize 29x29 "$OUTPUT_DIR/Icon-App-29x29@1x.png"
convert "$INPUT" -resize 58x58 "$OUTPUT_DIR/Icon-App-29x29@2x.png"
convert "$INPUT" -resize 87x87 "$OUTPUT_DIR/Icon-App-29x29@3x.png"
convert "$INPUT" -resize 40x40 "$OUTPUT_DIR/Icon-App-40x40@1x.png"
convert "$INPUT" -resize 80x80 "$OUTPUT_DIR/Icon-App-40x40@2x.png"
convert "$INPUT" -resize 120x120 "$OUTPUT_DIR/Icon-App-40x40@3x.png"
convert "$INPUT" -resize 120x120 "$OUTPUT_DIR/Icon-App-60x60@2x.png"
convert "$INPUT" -resize 180x180 "$OUTPUT_DIR/Icon-App-60x60@3x.png"
convert "$INPUT" -resize 76x76 "$OUTPUT_DIR/Icon-App-76x76@1x.png"
convert "$INPUT" -resize 152x152 "$OUTPUT_DIR/Icon-App-76x76@2x.png"
convert "$INPUT" -resize 167x167 "$OUTPUT_DIR/Icon-App-83.5x83.5@2x.png"
convert "$INPUT" -resize 1024x1024 "$OUTPUT_DIR/Icon-App-1024x1024@1x.png"

echo "图标生成完成！"
```

运行脚本：
```bash
chmod +x generate-icons.sh
./generate-icons.sh
```

1. 打开 `Assets.xcassets/AppIcon`

2. 在 Xcode 中选择你的设备
3. 运行应用
4. 查看主屏幕图标

## ⚠️ 常见问题

### Q1: 图标显示为白色方块

**原因**：图标有透明背景

**解决方案**：
1. 使用图片编辑软件（如 Photoshop、GIMP）
2. 添加白色或其他颜色背景
3. 导出为 PNG（无透明度）

### Q2: 图标模糊

**原因**：原始图片分辨率太低

**解决方案**：
- 使用至少 1024x1024 的高清图片
- 确保图片是 PNG 格式
- 避免从小图放大

### Q3: Xcode 显示警告

**常见警告**：
- "The app icon set has an unassigned child"
- "Missing required icon file"

**解决方案**：
- 确保所有必需尺寸都已添加
- 检查 `Contents.json` 文件是否正确

### Q4: 上传 App Store 时被拒

**原因**：
- 图标包含 Alpha 通道（透明度）
- 图标尺寸不正确
- 缺少 1024x1024 图标

**解决方案**：
- 使用在线工具重新生成
- 确保所有尺寸都正确
- 移除透明背景

## 🎨 图标设计建议

### 好的图标设计

✅ **简洁明了** - 一眼就能识别  
✅ **独特性** - 与其他应用区分开  
✅ **品牌一致** - 符合品牌形象  
✅ **适配性** - 在不同背景下都清晰  
✅ **可缩放** - 小尺寸也清晰可见  

### 避免的设计

❌ 过于复杂的细节  
❌ 太多文字  
❌ 使用照片（除非是品牌标志）  
❌ 过于相似的颜色  
❌ 透明背景
